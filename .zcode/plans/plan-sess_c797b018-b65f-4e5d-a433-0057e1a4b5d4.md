### 操作步骤

**第一阶段：拆分（纯搬文件 + 改名）**

1. `mkdir -p deepspace/src/ demo/src/bin/`
2. `git mv src/*.rs deepspace/src/`（所有物理/模拟模块）
3. `git mv deepspace/src/mission.rs deepspace/src/simulation.rs`（改名）
4. 调整 `deepspace/src/lib.rs`：移除 `pub mod app`，`simulation` 替代 `mission`
5. `git mv src/app.rs demo/src/app.rs`
6. `git mv src/bin/*.rs demo/src/bin/`
7. 写 3 个 `Cargo.toml`（workspace / library / demo）
8. 更新 demo 内的 `use crate::` → `use deepspace::`
9. 验证编译 + 测试

**第二阶段：render.rs — 3D 渲染辅助**

```rust
// render.rs — 在 macroquad 之上封装太空可视化原语

pub struct OrbitalCamera {
    pub target: Vec3,
    pub distance: f64,
    pub azimuth: f32,  // 水平旋转角
    pub elevation: f32,// 俯仰角
}

impl OrbitalCamera {
    pub fn update(&mut self);  // 处理鼠标拖拽/滚轮
    pub fn look_at(&self) -> (Vec3, Vec3);  // 返回 eye, target
}

/// 画行星（球体 + 颜色）
pub fn draw_planet(pos: Vec3, radius: f64, color: Color);

/// 画姿态 Gizmo（三个颜色轴表示飞船朝向）
pub fn draw_gizmo(pos: Vec3, orientation: &Quaternion, scale: f64);

/// 画轨迹线
pub fn draw_path(points: &[(Vec3, f64)], color: Color);

/// 画速度向量箭头
pub fn draw_velocity_arrow(pos: Vec3, vel: Vec3, scale: f64);
```

**第三阶段：nbody-viz.rs — N体沙盘可视化**

循环：加载 .scene → 每帧更新 N-body → 渲染行星球体 + 轨迹

```rust
// 伪代码
let mut world = SceneRuntime::new(&config);
let mut camera = OrbitalCamera::new();

loop {
    camera.update(&input);
    set_camera(&camera);
    
    world.step();
    for body in &world.sys.bodies {
        draw_planet(body.position, body.radius, body_color(body));
    }
    
    next_frame().await;
}
```

**第四阶段：rocket-viz.rs — 火箭发射可视化**

循环：加载任务配置 → 模拟火箭飞行 → 3D 实时展示

- 地球：蓝色大球体，自转
- 火箭：小标记 + 姿态 Gizmo（显示当前朝向）
- 轨迹：已飞路径（实线）+ 预测轨道（虚线）
- HUD 覆盖层：高度、速度、质量、级数、飞行时间
- 视角：可跟踪火箭或自由观察
